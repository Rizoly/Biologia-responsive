Archaea – Projeto Integrado de Biologia e Programação

Este repositório contém um site desenvolvido como projeto acadêmico para o Instituto Federal Farroupilha (IFFar) – Campus Santo Augusto, integrando as disciplinas de Programação e Biologia.

O tema foi Archaea (Arqueias), apresentando informações sobre suas características, estrutura, classificação, habitats, importância e diferenças em relação às bactérias, por meio de uma interface organizada, responsiva e de fácil navegação.

O objetivo do projeto foi aplicar conhecimentos de desenvolvimento web na criação de um material educativo, tornando o estudo das arqueias mais acessível e visualmente atrativo.

Tecnologias utilizadas
HTML5
CSS3
JavaScript
