Archaea – Projeto Integrador de Biologia e Programação

Este repositório contém um site desenvolvido como projeto acadêmico para o Instituto Federal Farroupilha (IFFar) – Campus Santo Augusto, integrando as disciplinas de Programação e Biologia.

O tema foi Archaea (Arqueias), apresentando informações sobre suas características, estrutura, classificação, habitats, importância ecológica e diferenças em relação às bactérias, por meio de uma interface organizada, responsiva e de fácil navegação.

O objetivo do projeto foi aplicar conhecimentos de desenvolvimento web na criação de um material educativo, tornando o estudo das arqueias mais acessível e visualmente atrativo.

Tecnologias utilizadas
HTML5
CSS3
Objetivos
Integrar os conteúdos das disciplinas de Programação e Biologia.
Desenvolver um site educativo sobre as arqueias.
Aplicar conceitos de estruturação e estilização de páginas web.
Proporcionar uma experiência de navegação simples, intuitiva e responsiva.

Este projeto foi desenvolvido para fins acadêmicos no IFFar – Campus Santo Augusto, demonstrando a aplicação prática do desenvolvimento web como ferramenta de apoio ao ensino de Biologia.
